Troubleshooting Common Issues

Page Order Problems

If booklet pages don't appear in correct order when folded, verify that you're using the correct printer settings for double-sided printing with short-edge binding.

Printing Alignment Issues

If pages don't align properly when folded, check printer settings and ensure 'Actual Size' scaling is selected to maintain proper page dimensions.

Content Layout Problems

If content doesn't display well in booklet format, consider adjusting margins and font sizes in the original document before creating the booklet.
