Troubleshooting Common Issues

Highlighting Precision

If highlights don't align properly with text, ensure you're drawing rectangles precisely over the intended text areas. Use the crosshair cursor for better accuracy.

Multiple Highlights

If you need to add multiple highlights, simply draw additional rectangles over different text sections. Each highlight will be preserved independently.

Save and Download

If highlighted documents don't save properly, ensure you click the 'Save PDF' button after completing all highlights. Check that popup blockers aren't preventing downloads.
