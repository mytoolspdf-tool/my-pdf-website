Troubleshooting Common Issues

Page Range Errors

If page range selection fails, verify that page numbers exist in the document and are specified correctly (e.g., 1, 3-5, 8).

Large File Processing

Very large PDF files may take longer to process. Ensure your browser has sufficient memory available and consider splitting in smaller batches if needed.

ZIP File Issues

If ZIP downloads fail, check your browser's download settings and ensure pop-ups are allowed for the splitting tool.
